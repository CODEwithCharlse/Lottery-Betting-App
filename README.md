# Lottery-Betting-App
1. The application must generate 6 UNIQUE random numbers (1-49). 2.
2. The application must then prompt the user how many numbers do they want to play.
3. The application must then prompt the user for their bet numbers (these must also be unique) and the amount they use to place a bet.
4. For each number won (guessed correctly), the winning amount is the double the number.
5. The total winning amount is calculated as the product of the winning amount * half the bet amount
6. The aplication must then display the generated lotto numbers and the user bet numbers both in an ascending order.
7. The number of correct guesses should be displayed with the correctly guessed numbers.
8. The betting amount should be displayed with the total winning amount
9. The application must continue running until the user chooses to stop
